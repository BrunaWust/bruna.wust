## 🌱 Sobre mim

Sou desenvolvedora de software na **Agro Baggio Máquinas Agrícolas**, concessionária John Deere localizada em Sorriso (Mato Grosso) — no coração do agronegócio brasileiro.

Minha trajetória é marcada por uma interseção pouco comum: **tecnologia aplicada diretamente ao campo**. Trabalho na ponte entre o que acontece na lavoura e o que pode ser medido, analisado e otimizado com dados. Não é só código — é entender o ciclo da safra, as dores do produtor, o que um mapa de solo precisa dizer.

Hoje meu foco está em **Machine Learning** para predição de cenários e otimização de processos agrícolas. Acredito que a agricultura de precisão ainda tem muito espaço para evoluir com software bem feito.

## 🚀 Projetos em destaque

### 🗺️ Geoanalytics Agrícola
Scripts em Python para processar shapefiles, interpolar dados de solo com técnicas geoestatísticas e gerar **mapas de aplicação variável**. O objetivo é reduzir desperdício de insumos e aumentar a precisão das decisões no campo.

`Python` `GeoPandas` `SciPy` `Matplotlib` `Shapefiles`

---

### 📊 Painel API Expert Connect
Dashboard com **auto-refresh e alertas** para monitoramento de consumo de APIs — construído com Flask + Dash. Usado internamente para acompanhar integrações com a plataforma John Deere Operations Center.

`Flask` `Dash` `Python` `REST API`

---

### 📲 Automatização de Relatórios via WhatsApp
Sistema que gera e envia automaticamente relatórios em PDF para produtores. 
`Python` `Wassenger API` `PDFKit` `Automação`
